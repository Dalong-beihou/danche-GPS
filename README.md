# GPS监控系统 - 完整项目文档

## 项目概述

一个基于Node.js和高德地图的GPS设备监控系统，专为校园共享单车智能定位与状态监测设计。支持实时轨迹追踪、设备状态管理、数据可视化、WebSocket实时推送、蓝牙定位和HTTPS公网访问。

**项目版本**: 1.1.0  
**技术栈**: Node.js, Express.js, WebSocket, HTML5, JavaScript, 高德地图API, ESP32  
**数据存储**: JSON文件存储  
**默认端口**: 5666

## 核心算法说明

### 1. GPS偏移校正
```javascript
// 北向偏移6米，东向偏移-125米
const GPS_OFFSET_NORTH_M = 6;
const GPS_OFFSET_EAST_M = -125;

function applyGpsOffset(lat, lng, valid) {
  if (!valid || lat == null || lng == null) return { lat, lng };
  const R = 6371000;
  const dLat = GPS_OFFSET_NORTH_M / R * (180 / Math.PI);
  const dLng = GPS_OFFSET_EAST_M / (R * Math.cos(lat * Math.PI / 180)) * (180 / Math.PI);
  return { lat: lat + dLat, lng: lng + dLng };
}
```

### 2. 距离计算（Haversine公式）
```javascript
function calculateDistance(lat1, lon1, lat2, lon2) {
  const R = 6371000; // 地球半径（米）
  const dLat = (lat2 - lat1) * Math.PI / 180;
  const dLon = (lon2 - lon1) * Math.PI / 180;
  const a = Math.sin(dLat/2) * Math.sin(dLat/2) +
            Math.cos(lat1 * Math.PI / 180) * Math.cos(lat2 * Math.PI / 180) * 
            Math.sin(dLon/2) * Math.sin(dLon/2);
  const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
  return R * c;
}
```

### 3. 速度平滑算法
```javascript
// 速度保持机制：当新速度为0但上次速度>0时，保持2500ms
if (parsed === 0 && prevValid && prevSpeed > 0 && now - nextTs < holdMs) {
  nextSpeed = prevSpeed; // 保持上次速度
} else {
  nextSpeed = parsed;    // 更新为新速度
  nextTs = now;
}
```
